# Changelog

## 0.1.0

Initial release.

- Template engine with `{{domain}}`, `{{date}}`, `{{counter}}`, `{{random}}`, and `{{mailDomain}}` placeholders
- Auto-detection of email fields via selectors and MutationObserver
- Shadow DOM overlay with inline popup for generating and filling addresses
- Context menu integration with preset support
- Keyboard shortcut (`Alt+M`) to trigger from focused field
- Options page with live template preview and preset management
- Toolbar popup showing current config
- Firefox build target via manifest patching
